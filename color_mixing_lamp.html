Color-Changing Lamp Arduino Project

link to code

Included Parts:

  • Arduino Uno Rev3
  • Tri-color LED
  • 3 phototransistors
  • 220 Ω resistor
  • 10K Ω resistor
  • Optional: colored gel strips (for testing)

This is project #4 of the Arduino Project Book. It creates a color-changing "lamp" from a tri-color LED that changes colors depending on external lighting conditions.

To use, set up the circuit as follows and upload the code to the arduino. The three phototransistors (one for R, G, and B) will begin sending their analog values to the arduino, which will translate the respective values into individual R, G, and B values to pass to the tri-color LED.

To test out changing only one color at a time, try covering the transistors with a red, green, or blue gel strip.

To test out changing all the colors, see how the LED reacts when the light around the phototransistors is dimmed or brightened.
