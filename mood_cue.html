Mood Cue Arduino Project

link to code

Included Parts:

  • Arduino Uno Rev3
  • Potentiometer
  • Servo motor & motor arm
  • 2- 100 µF capacitors

This is project #5 of the Arduino Project Book. Simple project for controlling the angle of a servo motor with a potentiometer.

To use, set up the circuit as follows and upload the code to the arduino. To change the position of the motor arm on the servo motor, turn the potentiometer in either direction. The servo should receive the analog values mapped to 0-180° and will turn the motor arm to face the respective position.

Called a "mood cue" because you can attach a paper arrow to the servo arm to point in different directions (such as a mood diagram).
