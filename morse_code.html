Morse Code Arduino Project

link to code

Included Parts:

  • Arduino Uno Rev3
  • 2 button switches
  • 3- 220 Ω resistors
  • LCD display (TC1602A-01T)
  • LED

The idea for this project came from an old school project I did that involved morse code inputs, and a project in the Arduino Starter Kit projects book for displaying magic 8 ball responses on an LCD screen.

The user can display messages translated from morse code inputs by holding down the transmission button (leftmost button) and entering dots and dashes via the input button (rightmost button). The letters are displayed to the user via the LCD screen, and via the serial monitor in the Arduino IDE.

For the sake of making the morse code input easier to use for someone who is not familiar with the morse code alphanumeric codes (such as myself), I have slowed down the input times such as follows:

 dot ('.')  5ms - 400ms input button press
 dash ('-')  400+ ms input button press
 completed letter  1500 - 3000ms, input button not pressed
 completed word (' ')   3000+ ms, input button not pressed
