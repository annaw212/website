Snake Game

The idea for this project came from an interest in learning C# and building a game, which is not something I had done before. The two versions of the game I created were C++ and C#. The C# language allows for developing a game with better graphics, whereas with C++ I was limited to creating a game that ran in the terminal and is quite limited graphically.

C++ Version

How to Play:

Run the game's executable using the command ./snake_game

  • Use the arrow keys or WASD to move the snake in the respective direction.
  • Food is depicted as *.
  • The snake's head is depicted as O, and its body is depicted using o for the length of the snake.
  • The borders are depicted using #.

The game will continue to run until the user chooses to quit the game using the q key, or runs into the wall, after which the user's final score will be displayed in the terminal.

C# Version

How to Play:

(Note: As far as I know, this game needs to be run using Visual Studio (NOT Visual Studio Code))

Once opened, there will be a button at the top of the software with a green triangle next to the name of the project. The solid green arrow will debug the code before running, and the green outline triangle will run the code normally.

  • The game's UI should open, displaying "Press any key to play."
  • Once a key is pressed, a countdown will begin, after which the snake will begin.
  • Use the arrow keys to control the snake's direction.
  • The game will continue until the snake runs into itself or a wall.
  • To grow the snake, eat the food (red circles) on the grid.
  • To pause the game, press the SPACE bar
  • To quit the game, press the q key
